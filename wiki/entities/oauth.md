# OAuth

OAuth 是一种开放标准的授权协议，旨在允许第三方应用在不暴露用户凭据（如密码）的前提下，获取对受保护资源的有限访问权限。在现代智能体架构中，OAuth 的作用至关重要，它充当了身份验证与资源访问之间的安全桥梁。通过将 OAuth 令牌存储在受保护的“安全保险库”中，系统能够实现凭据的集中管理与隔离，从而有效防止敏感信息泄露。

在 Claude 的运行环境中，OAuth 令牌的处理遵循严格的安全隔离原则。当智能体需要调用外部服务时，它不会直接接触或持有令牌。相反，系统通过一个专用的代理机制来处理这些请求：代理接收与当前会话关联的授权信息，并负责从安全保险库中安全地获取相应的 OAuth 凭据，进而完成对外部服务的调用。这种设计确保了辅助框架和沙盒环境永远无法获知或篡改任何凭据，极大地提升了系统的安全性。

### 在本视频中的角色
在报告描述的架构中，OAuth 是实现自定义工具安全集成的关键组件。它与 MCP（Model Context Protocol）协同工作，通过将令牌存储在沙盒之外的保险库中，确保了智能体在执行任务时既能获得必要的外部资源访问权限，又不会将敏感凭据暴露在 Claude 的上下文窗口或沙盒环境中。
