# MCP (Model Context Protocol)

MCP 是由 Anthropic 推动的一种标准化协议，旨在解决人工智能模型（如 [[Claude]]）与外部系统、工具及数据源之间的交互问题。在复杂的智能体架构中，MCP 的核心价值在于提供了一种安全且解耦的通信机制。通过该协议，智能体无需直接处理敏感的身份验证令牌或凭据，而是通过一个专用的代理层进行中转。当 Claude 需要调用自定义工具时，它会向该代理发送请求，代理随后从安全的保险库中获取相应的 OAuth 令牌或凭据，并代表智能体执行外部服务调用。这种设计确保了辅助框架和智能体本身永远不会接触到原始凭据，从而极大地降低了安全风险，并提升了系统的模块化程度。

### 在本视频中的角色
在报告所描述的架构中，MCP 扮演了“安全连接器”的角色。它被专门用于支持自定义工具的集成，通过将凭据存储在沙盒之外的安全保险库中，并利用专用代理进行凭据注入，MCP 成功地将智能体的逻辑执行与敏感的身份验证信息隔离开来。这种机制是实现[[托管智能体]]安全运行的关键基础设施之一，确保了在执行复杂任务时，即使智能体需要频繁调用外部 API，也不会导致凭据泄露。
